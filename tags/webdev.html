Pelican

Pelican is one of the most popular website generators available. It is written in python and comes with a lot of plugins to extend its functionality. Pelican is free/open source. The fetures that make it easy to use are: posts and pages are written in markdown. Custom html snippets can also be used. A large collection of themes and plugins available. write your own themes and plugins easily. Completely static output, can host anywhere.

Publishing a static website with pelican & github

Steps

  1. Create an empty repository where you'd host the generated content
  2. On your local storage, create a folder where your website contents will be stored and open a terminal.
  3. In the terminal type the following command to clone the repository git clone <repository address>
  4. This should create a .git folder in the current folder.
  5. Now we can start building our website by typing pelican-quickstart
  6. In the question whether you want to host on github, type Y.
  7. If this is your user page, accept yes otherwise no.
  8. To upload your website to github, use make github

Restoring repository

If by chance you end up changing one of the files online, make github would refuse to upload your files, in this case clone the repository in your output folder and move the .git folder back to your working directory.
