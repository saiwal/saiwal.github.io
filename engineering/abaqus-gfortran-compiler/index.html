Using gfortran compiler for user subroutines in ABAQUS 6.13

Published: Sun 25 September 2016
By Saiwal Krishna

In Engineering.

tags: abaqus

ABAQUS uses intel fortran compiler by default for compiling user subroutines. However a free alternative 'gfortran' can also be used. Here the main steps are outlined for using free gfortran compiler in linux to compile user subroutines for ABAQUS. This has been tested on ABAQUS 6.13, however this should also work for ABAQUS 6.10 and higher.

This method has been tested and verified on ABAQUS 6.13 running on xubuntu 16.04.1

  1. Make sure gfortran is installed. If not install it by running

     sudo apt-get install gfortran

  2. Locate the abaqus_v6.env file. Its located in the installation folder under 613/6.13-4/SMA/site/

  3. rename it to ~abaqus_v6.env. This will server as a backup file incase something goes wrong.

  4. Replace it with this file.

  5. Be careful to replace the last few lines particularly abaquslm_license_file=' ' about your licence information with the information in your backup file.

  6. Run the following command to verify:

    abaqus verify -user_std

The following will confirm a working fortran compiler.
