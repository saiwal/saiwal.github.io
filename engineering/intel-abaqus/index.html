Linking Intel fortran compiler to ABAQUS and using UMAT

Published: Mon 30 June 2014
By Saiwal Krishna

In Engineering.

tags: abaqus

Prerequisites: sudo apt-get install rpm build-essential sudo apt-get install libstdc++6 sudo apt-get install ia32-libs

Download the Intel® Fortran Composer XE 2013 for Linux from https://software.intel.com/en-us/non-commercial-software-development You will recieve activation code via email.

Prefer the online installer as it is smaller in size and you can choose what components to install.

The installer is of the form l_fcompxe_online_2013_sp1.3.174.sh Execute the file from terminal with root privileges. Install the compiler. you may choose not to install other components.

Add compiler path to ~/.bashrc by adding following lines at the end:

PATH=”/opt/intel/composer_xe_2013_sp1.3.174/bin/intel64:$PATH”
export PATH
LD_LIBRARY_PATH=”/opt/intel/lib/intel64:$LD_LIBRARY_PATH”

then run the following,

source .bashrc

check for the path, change it appropriately according to your system. the intel64 at the end of lines would change to intel32 for 32 bit systems.

type the command ifort in the terminal and check if compiler is recognized.

to submit a job to abaqus using command line use the following syntax:

abaqus job=(job name) inp=(input file name) user=(user subroutine)
