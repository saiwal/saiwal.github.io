SymPy for symbolic calculations

Published: Fri 19 June 2015
By Saiwal Krishna

In Engineering.

tags: Linux python

Continuing our journey towards python for research, we take a look at sympy, a library for symbolic math. Installation on a linux system is simple through apt-get. The most efficient way to use this package would be through ipython notebook. Lets take a look. Start a ipython notebook by typing ipython notebook at the terminal. The browser should open a new window. Click on new notebook to open a new worksheet. This is what it should look like: You can start typing python code in the line [1] and press shift+Enter to execute it.
