    Continuing our journey towards python for research, we take a look at sympy, a library for symbolic math. Installation on a linux system is simple through apt-get. The most efficient way to use this package would be through ipython notebook. Lets take a look. Start a ipython notebook by typing ipython notebook at the terminal. The browser should open a new window. Click on new notebook to open a new worksheet.
