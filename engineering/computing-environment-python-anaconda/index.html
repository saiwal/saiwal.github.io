  • Setting up Computing environment on ubuntu using python
    Tue 05 January 2016 Engineering python / research / linux

    Continuum Analytics provides Anaconda, a platform which provides many python libraries bundled together as a single environment package. The bundle can be downloaded from the website for different OS. Once installed you can set it up to use as a separate environment for computing without interfering with the already installed python modules on your system.

    To activate the environment, we use the commmand

    cd anaconda3/
source ~/bin/activate ~/anaconda3/

    Assuming the the anaconda package has been installed at ~/anaconda3/

    To deactivate after you're done, use :

    source deactivate

    The package also includes spyder which provides a very helpful interface to code and inspect variables in matlab style.
